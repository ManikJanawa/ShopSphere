# ShopSphere - Multi Vendor E-Commerce System

## Technology Used
- Java Swing
- MySQL
- JDBC

## Modules
- Admin Module
- Vendor Module
- Customer Module

## Features
- User Authentication
- Product Management
- Product Search
- Shopping Cart
- Checkout
- Cash on Delivery
- Billing System
- Order Management
- Inventory Management

## Project Description
ShopSphere is a Java Swing and MySQL based multi-vendor e-commerce application. Customers can browse products, add items to cart, place orders and track order status. Vendors manage products and inventory, while Admin manages the complete system.

## How to Run
1. Import the MySQL database.
2. Update database connection details.
3. Run Login.java file.
